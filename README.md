## üöÄ Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Set Up Database

Create a `.env.local` file in the project root:

```bash
DATABASE_URL=your_neon_postgres_connection_string
```

Then push the schema:

```bash
npm run db:push
```

### 3. Seed Data (Optional)

```bash
npm run seed
```

### 4. Start the App

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
